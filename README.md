# Sensor-de-Umidade-de-Gr-os-com-ESP32
Este projeto consiste em um sistema embarcado de monitoramento de microclima e umidade interna para grãos. Desenvolvido com ESP32, o dispositivo utiliza uma abordagem de fusão de dados entre sensores capacitivos e higrométricos para fornecer uma leitura mais precisa da umidade final do grão.
